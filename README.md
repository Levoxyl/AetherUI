# 💻 Hacker Visual

A sleek, lightweight Python terminal application designed to bring a high-tech, cyberpunk "hacker" aesthetic straight to your desktop. Built using Object-Oriented Programming (OOP), Hacker Visual features dynamic terminal rendering, real-time visual flows, and fully customizable themes.

## 🚀 Features

* **Custom Themes:** Fully structured theme support, featuring high-contrast palettes, smooth terminal interfaces, and custom color mappings.
* **Dynamic UI Elements:** Real-time stream animations and responsive terminal layouts.
* **Clean OOP Architecture:** Built from the ground up with modular, easily extensible Python classes.
